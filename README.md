# simple-chat-app
Simple chat application with Socket.io &amp; Nodejs &amp; MongoDb &amp; Mongoose

##
1- run
```bash
npm install 
```

2- make .env file with below format :

```bash
 SERVER_IP =
 DB_CONNECTION = 
```

save the file 

3- Set your db connection string.
 *If you wanna run it on localhost , leave SERVER_IP  as is.
